# IPGTT Analysis Tool

Automated IPGTT report generation tool.

## Features

- HI/LO glucose conversion
- AUC calculation
- Mean ± SD report
- Mean ± SE report
- Error Log sheet
- Automatic chart generation

## Input

Excel file containing:

- Group
- Mouse_ID
- Time points

## Output

Excel report including:

- Input_Copy
- Error_Log
- Processed_Wide
- Mean_SD_AUC
- Mean_SE_AUC
